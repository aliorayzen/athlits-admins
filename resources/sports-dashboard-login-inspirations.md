# Sports & Dashboard Login — Inspiration Board

Curated references for the Arena Admin login page, focused on **sports booking platforms, SaaS dashboards, and premium dark split-screen** patterns. Click each link to browse; each entry notes what to borrow for Arena's dark-luxury aesthetic (teal accent, gold micro-highlights, 48/52 split panel).

> Companion to `login-inspirations.md` (general patterns) and `dashboard-inspirations.md` (full-dashboard UX).

## 7. Arena-Specific Takeaways

After scanning these references, here are the gaps our current login doesn't yet cover that could be worth pulling in:

| #   | Opportunity                                                                          | Reference                            | Priority                               |
| --- | ------------------------------------------------------------------------------------ | ------------------------------------ | -------------------------------------- |
| 7.1 | **Mouse-parallax on hex grid / glow orbs**                                           | Premium dark inspiration (#3.1–#3.5) | Medium — signature moment              |
| 7.2 | **"Last login" metadata** below status pill on left panel                            | SaaS admin patterns (#2.1)           | Low                                    |
| 7.3 | **Keyboard-shortcut hint** ("⏎ to submit") subtle under button                       | Linear aesthetic (#5.2)              | Medium                                 |
| 7.4 | **Brand-specific illustration** on left panel (stylized sports court/arena top-down) | Sports dashboards (#1.1, #1.7)       | Large effort, high signature value     |
| 7.5 | **Biometric / passkey CTA** as secondary auth option                                 | OTP UI kits (#4.1)                   | Medium — forward-looking               |
| 7.6 | **Subtle video/looping background** on left panel (court time-lapse)                 | Sports analytics apps (#1.5)         | Large effort — only if we commission   |
| 7.7 | **Skeleton-shimmer empty state** on dashboard post-login                             | Stripe/Linear (#5.1, #5.2)           | High — applies to dashboard, not login |

## How to use this file

1. **Open links in tabs.** Dribbble/Pinterest/Figma-community links are all direct-browsable without accounts (account helps for favoriting).
2. **Screenshot what you like** into `resources/inspo/` — there's already a folder there for captured references.
3. **Flag 2–3 signature ideas** you want to pursue; we'll scope them into a Top-Design v2 pass on login, or roll them into the dashboard (Step 6 in the skills plan).
4. **Companion docs:**
   - `resources/login-inspirations.md` — general pattern catalog (glassmorphism, neon, etc.)
   - `resources/dashboard-inspirations.md` — post-login dashboard references
   - `resources/inspo/` — raw screenshots you've already captured
